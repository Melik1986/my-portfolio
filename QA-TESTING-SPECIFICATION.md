## 🎯 Конкретные проблемы для исследования

### 🚨 Критические проблемы

#### 1. Неверный порядок инициализации
**Симптомы:**
- Анимации запускаются до завершения Loader
- ScrollTrigger не работает без ресайза окна
- Множественная инициализация компонентов

**Места для проверки:**
```typescript
// GSAPInitializer.ts
// Проверить порядок:
// 1. window.onload
// 2. document.readyState === "complete"
// 3. Loader завершен
// 4. ScrollTrigger.refresh()
```

#### 2. Проблемы с ScrollTrigger
**Симптомы:**
- Триггеры не срабатывают
- Анимации "прыгают"
- Неправильные границы срабатывания

**Места для проверки:**
```typescript
// useScrollSmoother.ts
// Проверить:
// - Инициализация после DOM ready
// - Правильные селекторы
// - Обновление после изменений
```

#### 3. 3D аватар не загружается
**Симптомы:**
- Модель не отображается (рендерится только canvas без модели)
- Ошибки Three.js в консоли
- Проблемы с WebGL контекстом
- Canvas элемент присутствует в DOM, но 3D-модель не загружается

**Места для проверки:**
```typescript
// 3DAvatar.tsx
// Проверить:
// - Асинхронную загрузку модели
// - Обработку ошибок
// - Инициализацию Three.js
// - Корректность путей к 3D-ресурсам
// - Состояние загрузки модели
```

### ⚠️ Средние проблемы

#### 4. Навигация работает нестабильно при изменении размера экрана
**Симптомы:**
- При резкой смене размера экрана навигация перестает работать
- Клики по ссылкам не приводят к переходу к секции
- Кнопка `AnchorButton` пропадает после резайза окна
- Неправильная прокрутка к секциям после изменения viewport
- Проблемы с активными состояниями навигации
- ScrollTrigger теряет корректные границы после resize

**Компоненты для дополнительной проверки:**
```typescript
// AnchorButton компонент
// Путь: src/lib/ui/AnchorButton/
// Проверить:
// - Обработку resize событий
// - Пересчет позиций после изменения размера
// - Условную логику отображения компонента
// - Связь с ScrollTrigger и пересчет координат
```

#### 5. Анимации карточек глючат
**Симптомы:**
- Карточки "застревают" в промежуточных состояниях
- Неправильные тайминги
- Проблемы с hover эффектами

### Unit Test Template
```typescript
// tests/unit/ComponentName.test.ts
import { render, screen } from '@testing-library/react';
import { ComponentName } from '@/path/to/component';

describe('ComponentName', () => {
  beforeEach(() => {
    // Подготовка перед каждым тестом
  });

  afterEach(() => {
    // Очистка после каждого теста
  });

  test('должен рендериться без ошибок', () => {
    // Arrange
    const props = { /* тестовые пропсы */ };
    
    // Act
    render(<ComponentName {...props} />);
    
    // Assert
    expect(screen.getByRole('...')).toBeInTheDocument();
  });

  test('должен обрабатывать клик корректно', async () => {
    // AAA паттерн
  });
});
```

### E2E Test Template
```typescript
// tests/e2e/feature.spec.ts
import { test, expect } from '@playwright/test';

test.describe('Feature Name', () => {
  test.beforeEach(async ({ page }) => {
    await page.goto('/');
    // Ждем готовности страницы
    await page.waitForLoadState('networkidle');
  });

  test('должен выполнять основной сценарий', async ({ page }) => {
    // Arrange: подготовка данных
    
    // Act: выполнение действий
    await page.click('[data-testid="button"]');
    
    // Assert: проверка результата
    await expect(page.locator('[data-testid="result"]')).toBeVisible();
  });
});
```
